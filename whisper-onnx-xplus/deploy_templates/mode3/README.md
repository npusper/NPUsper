# deploy_templates/mode3

This directory records the working deploy-mode3 configuration from the X Plus
laptop at:

```text
%USERPROFILE%\Desktop\deploy_mode3
```

The full deploy directory is generated by:

```bash
whisper_env/bin/python compile/package_nk_full_deploy.py \
  --model base --target xplus --buckets all --with-30s \
  --stage-dir ./artifacts/deploy_mode3
```

Only the small reference `config.json` is committed here. Generated EPContext
wrappers and QNN context binaries are rebuilt by the compile/package pipeline.
